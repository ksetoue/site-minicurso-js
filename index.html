Aprendendo a interfacear html, css e js

Javascript é uma linguagem interpretada, desenvolvida para ser usada no lado cliente. O browser lê e interpreta o código enquanto ele é executado.

Embora o Javascript tenha sido desenvolvido com foco no lado cliente do desenvolvimento Web, hoje é possível usá-lo em aplicações mais robustas.

Isso é possível graças a plataformas como o Node.js
